# Roadmap

## Phase 1: API Foundation

- Finalize domain boundaries and DTO conventions.
- Add EF Core migrations.
- Implement authentication, authorization, and account protections.
- Add validation, audit fields, and structured error responses.

## Phase 2: Next.js Website

- Build public pages for programs, events, store, donate, testimonials, and contact.
- Connect frontend API client to real backend endpoints.
- Add accessible forms and loading/error states.

## Phase 3: Admin Dashboard

- Add role-based admin workflows.
- Manage athletes, events, products, testimonials, documents, and submissions.
- Add audit views and operational reporting.

## Phase 4: Payments and Donations

- Integrate payment provider workflows.
- Handle webhooks, receipts, refunds, and donation records.
- Harden payment logging and reconciliation.

## Phase 5: Mobile and iOS

- Define mobile API contracts.
- Build mobile-first parent and coach workflows.
- Evaluate native iOS or cross-platform implementation based on club needs.
